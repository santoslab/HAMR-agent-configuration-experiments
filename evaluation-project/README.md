# Evaluation Project

This folder holds evaluation HAMR projects used to test and improve the training material in `@hamr-claude-training`. Claude is asked to consult the training material and then perform development tasks on the evaluation project here. Based on situations where Claude doesn't perform well, the training material is updated to provide better context.

When an evaluation project is complete, its artifacts are promoted to a training example in `hamr-claude-training/examples/` so future developers benefit from the additional context.
